# Changelog

All notable changes to **business-data-automator** are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [1.2.0] - 2026-06-05

### Added
- **Dynamic shields.io badges** in README (version, license, stars, last-commit)
- **GitHub Topics** for discoverability (`automation`, `business-intelligence`, `ai-skill`, `cline-skill`, `data-automation`, `revenue-forecasting`)
- **CI workflow** (`.github/workflows/ci.yml`) for Markdown lint + link check + YAML frontmatter validation
- **CHANGELOG.md** following Keep a Changelog format

### Changed
- Replaced static version badge with live `github/v/release` shield

## [1.1.0] - 2026-06-05

### Added
- Structured YAML frontmatter (`name`, `description`, `version`, `tags`)
- **ML-Assisted Forecasting** section (time-series, regression, ensemble methods)
- **Secrets Management** patterns (env vars, vaults, rotation, audit trails)
- **MCP (Model Context Protocol) & A2A (Agent-to-Agent)** integration notes
- **Comprehensive Troubleshooting Guide** with diagnostic flowcharts
- Expanded **Anti-Patterns** section with concrete fixes

### Changed
- Clearer capability boundaries and trigger phrases
- Better cross-references between sections

## [1.0.0] - 2026-06-05

### Added
- Initial release
- `SKILL.md` covering data profiling, cleaning patterns, schema design, dashboard architecture, automation recipes, client-type playbooks, pricing, anti-patterns
- `README.md`, `CONTRIBUTING.md`, `LICENSE`
- First tagged release
